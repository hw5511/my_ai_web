== 나만의 커뮤니티 기획서 ==

🎯 커뮤니티 주제
- 게임 커뮤니티 

🎨 디자인 스타일
- 사이트 이름: "🎮 Game cube"
- 색상 테마: 어두운 다크테마 배경에 녹색 네온컬러테마
- 분위기: 미래적이고, 기하학적인 요소

📱 특별한 기능 아이디어
- 좋아요 기능

🖼️ 페이지별 특징
1. 로그인페이지
	-> 아이디/비밀번호 입력란
	-> 회원가입 버튼
	-> 로그인 버튼 -> 로그인 성공시 메인페이지로 이동
2. 회원가입 페이지
	-> 아이디/비밀번호 입력란
	-> 회원가입 버튼
	-> 회원가입에 성공하면 로그인페이지로 돌아오기
3. 메인페이지 
	-> 네비게이션 바
		- 로고
		- 프로필영역
			- 닉네임(아이디표시)
			- 로그아웃버튼
		- 글쓰기 버튼
	-> 메인콘텐츠 영역
		- 게시물 목록(리스트 방식)
			- 게시물 
				- 제목
				- 작성자
				- 댓글수
				- 좋아요 수
				- 작성 시간
4. 게시물 상세페이지
	- 뒤로가기 버튼 
	- 게시물 영역
		- 제목/작성자/작성시간
		- 게시물 내용
		- 좋아요버튼/좋아요수
	- 댓글 영역
		- 댓글입력란
		- 댓글 목록
			- 댓글
				- 댓글 내용
				- 댓글 작성자
				- 작성시간
5. 게시물 작성페이지
	- 제목 입력란
	- 내용입력란
	- 게시물등록버튼
	- 뒤로가기버튼
