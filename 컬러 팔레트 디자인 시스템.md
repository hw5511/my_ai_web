## 🎨 컬러 사용 가이드

### 1. Primary Color (크림슨 레드 #C41E3A)
| 사용 위치 | 용도 |
|-----------|------|
| 헤더/로고 | 브랜드 아이덴티티 강조 |
| CTA 버튼 | 주요 행동 유도 버튼 |
| 중요 텍스트 | 타이틀, 강조 문구 |
| 아이콘 | 핵심 기능 아이콘 |

```css
/* 사용 예시 */
.cta-button {
  background-color: var(--color-primary);
  color: var(--color-text-inverse);
}

.page-title {
  color: var(--color-primary);
}
```

### 2. Secondary Color (라일락 퍼플 #9B7BB8)
| 사용 위치 | 용도 |
|-----------|------|
| 보조 버튼 | Secondary CTA |
| 태그/뱃지 | 카테고리 표시 |
| 호버 효과 | 링크 호버 색상 |
| 장식 요소 | 배경 패턴, 아이콘 |

```css
/* 사용 예시 */
.tag {
  background-color: var(--color-secondary-light);
  color: var(--color-secondary-dark);
}

a:hover {
  color: var(--color-secondary);
}
```

### 3. Background Colors (배경색)
| 색상 | 사용 위치 |
|------|-----------|
| `--color-bg-primary` | 페이지 전체 배경 |
| `--color-bg-secondary` | 섹션 구분, 푸터 |
| `--color-bg-tertiary` | 카드, 모달, 입력 필드 |
| `--color-bg-accent` | 하이라이트 영역, 배너 |

```css
/* 사용 예시 */
body {
  background-color: var(--color-bg-primary);
}

.card {
  background-color: var(--color-bg-tertiary);
  border: 1px solid var(--color-border-light);
}

.highlight-section {
  background: var(--gradient-background);
}
```

### 4. Text Colors (텍스트)
| 색상 | 사용 위치 |
|------|-----------|
| `--color-text-primary` | 본문, 제목 |
| `--color-text-secondary` | 부제목, 레이블 |
| `--color-text-muted` | 설명, 캡션 |
| `--color-text-light` | placeholder, 비활성 |

```css
/* 사용 예시 */
h1, h2, h3 {
  color: var(--color-text-primary);
}

p {
  color: var(--color-text-secondary);
}

.caption {
  color: var(--color-text-muted);
}
```

## 📊 색상 대비 검증 (WCAG 2.1)

| 조합 | 대비율 | 등급 | 용도 |
|------|--------|------|------|
| `#C41E3A` on `#FDF5F3` | 6.2:1 | AA ✅ | 일반 텍스트 |
| `#2D2D2D` on `#FDF5F3` | 12.5:1 | AAA ✅ | 본문 텍스트 |
| `#FFFFFF` on `#C41E3A` | 5.8:1 | AA ✅ | 버튼 텍스트 |
| `#9B7BB8` on `#FDF5F3` | 3.8:1 | AA (Large) ⚠️ | 큰 텍스트만 |
| `#6B6B6B` on `#FDF5F3` | 5.1:1 | AA ✅ | 보조 텍스트 |

> ⚠️ 라일락 퍼플(`#9B7BB8`)은 18px 이상의 큰 텍스트에만 사용 권장

## ✨ 추가 참고사항

1. **색상 일관성**: 전체 프로젝트에서 CSS 변수만 사용하여 일관성 유지
2. **접근성**: 텍스트와 배경 간 최소 4.5:1 대비율 준수
3. **브랜드 확장**: Primary/Secondary 색상을 기준으로 파생 색상 생성
4. **테스트**: 다양한 디바이스와 조명 환경에서 색상 테스트 권장
